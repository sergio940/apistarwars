# Star Wars API

API de Star Wars hecha con Node.js y Express. Permite consultar personajes de Star Wars.

## Endpoints

- `GET /characters` → devuelve todos los personajes  
- `GET /characters?id=1` → filtra por id  
- `GET /characters?name=luke` → filtra por nombre  
- `GET /characters/:id` → devuelve personaje por id
